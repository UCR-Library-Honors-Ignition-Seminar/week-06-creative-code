## Before You Start

Play a Bitsy game or look at p5.js sketches before you build your own — it will change what you make. See the **Examples to Explore** section in [README.md](README.md). This should take 10–15 minutes and is not optional.

---

## Project

Choose **one** of the following:

---

### Option A: p5.js Sketch

Create a generative sketch using the [p5.js Web Editor](https://editor.p5js.org).

**Requirements:**
- **Generative** — produces different output each run, changes over time, or responds to mouse/keyboard input
- **Intentional** — has a concept you can describe in one sentence (a mood, a system, a question — not just "random shapes")
- Uses at least **3 different p5.js functions** beyond the starter template
- Runs without errors

**To submit:**
1. Save your sketch (create a free account to save, or download)
2. Click **File > Share** → copy the "Sketch" link
3. Create `links.md` in your repo and paste the link there

**Step-by-step help:** [INSTRUCTIONS-P5.md](INSTRUCTIONS-P5.md)

---

### Option B: Bitsy Game

Build a complete game using the [Bitsy editor](https://make.bitsy.app).

**Requirements:**
- At least **3 rooms** connected by exits
- At least **2 sprites with dialogue** — characters, objects, or environmental text
- A clear sense of beginning and somewhere to arrive (ending tile, or a final room)
- A title (set in Game > Title)

**To submit:**
1. Click **Game > Save game** → a `.html` file downloads
2. Upload the `.html` file to your GitHub repo

**Step-by-step help:** [INSTRUCTIONS-BITSY.md](INSTRUCTIONS-BITSY.md)

## Reflection

Write a **~300-word reflection** in `reflection.md` using the [reflection template](reflection-template.md).

**This week's critical prompt (Part 3 of the template):**

> *How does working within the constraints of p5.js or Bitsy change what you make? What would be impossible to express in this tool — and does that limitation matter?*

Think about: what the tool made easy and what it made impossible; a specific decision the constraints forced you to make; whether the limitation felt creative or frustrating — or both.

---

## Submission

1. Add your project file(s) — `links.md` for p5.js, `.html` file for Bitsy
2. Create `reflection.md` using the reflection template
3. Commit your changes with a short message describing what you added
4. Submit your repo URL on Canvas
